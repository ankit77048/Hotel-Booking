# Hotel booking System

Act as Software full Stack Developer, Develop a responsive, modern Hotel Booking System with a scalable backend architecture, secure authentication, and interactive UI animations.  
It allows users to search hotels, view the price and details of the room, check the room availability, book hotel rooms securely, pay online, and receive booking confirmation through emails.

## Features should include:

1. User-friendly with mobile, tablet and desktop UI for modern devices.
2. List and filter of hotel prices.
3. Hotel ratings and locations.
4. List of amenities, Gallery of room pictures, Availability of room.
5. Secure login and registration system using JWT authentication.
6. Reservation system that offers check-in/out options and a bookings overview.
7. Include integrating payment with Stripe or Razor pay.
8. Users can see the booking history.
9. Users can cancel and book hotel rooms.
10. Get confirmation of booking or cancel through email.
11. Validating forms on the client and server.
12. Secure REST APIs using Node.js and Express/Next.js
13. Connect with a PostgreSQL database.
14. Error handling, loading states and structured json responses.
15. Security measures: sanitizing the inputs, limiting number of requests, and using environment variables
